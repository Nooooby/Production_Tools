# Production Operations Dashboard

这是一个生产运营仪表板项目，用于管理和分析 Excel 数据。

## 项目结构

```
Production_Operations_Dashboard/
├── data/                 # Excel 文件和数据存放处
│   └── (在这里放你的 Excel 文件)
├── scripts/              # Python 脚本存放处
│   └── (用来处理 Excel 的脚本)
├── docs/                 # 项目文档
│   └── (项目说明和笔记)
└── README.md            # 项目说明（这个文件）
```

## 使用步骤

1. **复制 Excel 文件**：将你的 Excel 文件放到 `data/` 文件夹
2. **创建脚本**：在 `scripts/` 文件夹创建 Python 脚本来处理数据
3. **记录笔记**：在 `docs/` 记录项目进展和重要信息
