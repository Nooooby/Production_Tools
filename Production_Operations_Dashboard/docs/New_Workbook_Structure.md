## 🔑 关键表说明

### 最关键的 5 个表

#### 1️⃣ 05_Daily_Orders - 订单主表
- **用途**: 每日订单输入和追踪，从 00_SKU_Master 调用 SKU 数据
- **频率**: 每天更新
- **重要性**: ⭐⭐⭐
- **数据来源**: 从 00_SKU_Master 获取 SKU 参考数据（1,315 处引用）
- **依赖关系**: 调用 00_SKU_Master；所有其他表都参考此表
- **关键列**: SKU (来自 00_SKU_Master), Cases Ordered, Cases Added On, Cases Completed

#### 2️⃣ 00_SKU_Master - 产品参考
- **用途**: 所有产品信息，为 05_Daily_Orders 提供 SKU 数据
- **频率**: 定期更新（产品变化时）
- **重要性**: ⭐⭐⭐
- **数据流**: 被 05_Daily_Orders 大量调用（1,315 处），也被其他表引用
- **依赖关系**: 被 05_Daily_Orders 和其他表引用
- **关键列**: SKU (主键), Customer, Product_Group, Avg_Case_Weight

#### 3️⃣ 06_Resource_Plan - 原料计划
- **用途**: 计算每天需要的原材料
- **频率**: 每天自动更新
- **重要性**: ⭐⭐⭐
- **依赖关系**: 依赖 05_Daily_Orders 和 00_Yield_Rates
- **关键列**: Cases of Raw Required

#### 4️⃣ 07_Labor_Calc - 工时计算
- **用途**: 预测每天需要的工作时间
- **频率**: 每天自动更新
- **重要性**: ⭐⭐⭐
- **依赖关系**: 依赖 05_Daily_Orders
- **关键列**: STD, Total Earned Hr

#### 5️⃣ 12_Executive_Dash - 执行仪表板
- **用途**: 管理层关键指标
- **频率**: 实时更新
- **重要性**: ⭐⭐⭐
- **依赖关系**: 综合所有关键表
- **关键列**: 各种 KPI

## ✨ 使用建议

### 日常操作

```
早上:
  1. 打开 v39_Normalized.xlsx
  2. 检查 05_Daily_Orders (新订单?)
  3. 查看 06_Resource_Plan (今天需要多少原料?)
  4. 查看 07_Labor_Calc (今天工时充足吗?)

中午:
  5. 更新 13_Progress_Track (当前进度)
  6. 检查 12_Executive_Dash (管理层关键指标)

下午:
  7. 准备 11_Daily_Report (日报告)
  8. 预览 15_5Day_Forecast (未来5天计划)
```

### 周期性任务

```
周末:
  □ 审核 14_Weekly_Plan (下周计划)
  □ 检查 01_Cages_Plan (分割计划)
  □ 更新 00_SKU_Master (如有产品变化)

月末:
  □ 汇总 11_Daily_Report
  □ 分析趋势
  □ 为 00_Yield_Rates 做调整
```
